# Планировщик совместных мероприятий

## Описание
Веб-приложение для организации встреч с друзьями. Позволяет создавать мероприятия, приглашать участников, голосовать за время и место, вести общий чек-лист.

## Функции
- Регистрация по номеру телефона
- Создание мероприятий
- Приглашение участников по телефону
- Голосование за время и место
- Общий чек-лист
- Список участников

## Технологии
- Python 3
- Flask
- SQLite
- HTML/CSS
